🚀 FinUP – Autonomous Financial Intelligence Platform

FinUP is an AI-powered Personal Finance Management System that analyzes bank statements, categorizes expenses using Machine Learning, tracks savings goals, detects anomalies, forecasts future spending, and generates intelligent financial insights.

---

📌 Features

🔐 Authentication
- User Registration
- Secure Login
- Password Hashing
- Session Management
- Multi-user Support

📊 Financial Dashboard
- Total Income
- Total Expenses
- Net Savings
- Savings Rate
- Cash Flow Visualization
- Category-wise Analytics

🏦 Bank Statement Analysis
- CSV Statement Upload
- Excel Statement Upload
- Automatic Transaction Extraction
- Multi-bank Statement Support

🤖 AI Expense Categorization
- Food
- Shopping
- Entertainment
- Utilities
- Health
- Transport
- Investment
- Education
- Transfer
- Income

Powered by:
- TF-IDF Vectorization
- Logistic Regression
- Label Encoding

📉 Anomaly Detection
- Detects unusual transactions
- Flags suspicious spending
- Financial risk monitoring

🎯 Goal Tracking
- Create Savings Goals
- Monitor Progress
- Achievement Tracking
- Savings Rate Analysis

💰 Investment Monitoring
- Detect SIP Transactions
- Mutual Fund Tracking
- Investment Summary

📈 Spending Forecast
- Future Expense Prediction
- Budget Planning
- Monthly Forecast Analysis

📄 Smart Reports
- Historical Report Storage
- Excel Export
- Category Summary
- Income & Expense Analysis
